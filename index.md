# BrainTax

**BrainTax** is a short-form video channel about the psychology behind money decisions — the cognitive biases, mental shortcuts, and emotional patterns that shape how people spend, save, and invest.

Episodes are published daily on:

- TikTok — [@braintaxhq](https://www.tiktok.com/@braintaxhq)
- YouTube Shorts
- Instagram Reels
- Facebook Reels

## About this site

This site serves two purposes:

1. A public home page for the BrainTax channel.
2. Documentation of the self-hosted publishing tool used to operate the `@braintaxhq` TikTok account via the TikTok Content Posting API. The tool runs on the operator's own infrastructure and publishes only to accounts owned by the operator. See the [Privacy Policy](./privacy/) for details on API scopes and data handling.
